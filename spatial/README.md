# Contains file for the analysis of spatial data 

- ***visium:*** This folder contains all the files required for the visium part of the analysis except commot.
- ***stereoseq:*** This folder contains all the files required for the steroeseq part of the analysis except commot.
- ***commot.yml:*** This is an environment often used by both modality analysis.
- ***pipeline_commot:*** This folder contains the workflow to run commot on both stereoseq and visium.
